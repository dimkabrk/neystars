Правила пользования сервисом NeyStars

Дата вступления в силу: 4 июля 2025 года

1. Общие положения

1.1. NeyStars — сервис по продаже Telegram Stars (далее — "Звёзды") для поддержки создателей контента.

1.2. Используя сервис, вы соглашаетесь с настоящими Правилами и подтверждаете, что ознакомились с ними.

2. Условия покупки

2.1. Для совершения покупки необходимо:

  • Иметь действующий аккаунт Telegram
  • Указать корректный @username в форме заказа
  • Совершить оплату выбранного пакета Звёзд

2.2. В комментарии к платежу обязательно укажите ваш @username, иначе средства будут зачислены как донат.

3. Процесс покупки

3.1. Покупка включает следующие этапы:

  • Выбор пакета Звёзд из доступных вариантов
  • Указание Telegram @username
  • Оплата через интегрированную платежную систему
  • Мгновенное зачисление Звёзд на ваш аккаунт
!

Важно: Все транзакции являются окончательными. Возврат средств за приобретённые Звёзды не производится.

4. Обязанности сторон

4.1. Сервис NeyStars обязуется:

  • Обеспечивать мгновенную доставку оплаченных Звёзд
  • Поддерживать актуальные цены, указанные на сайте
  • Гарантировать конфиденциальность данных пользователей

4.2. Пользователь обязуется:

  • Предоставлять достоверную информацию при оформлении заказа
  • Использовать сервис только в законных целях
  • Не пытаться обойти систему безопасности

5. Ограничения

5.1. Запрещается:

  • Использовать сервис для мошеннических операций
  • Покупать Звёзды с использованием украденных платежных средств
  • Перепродавать приобретённые Звёзды
  • Использовать сервис лицам младше 13 лет

6. Ответственность

6.1. Сервис не несёт ответственности за:

  • Некорректное указание @username пользователем
  • Блокировку аккаунта Telegram со стороны мессенджера
  • Изменения в политике Telegram относительно Stars

По всем вопросам обращайтесь: support@neystars.ru
